# Maze Solver

Given a maze (represented by an array of strings), a wall (represented by a string), a starting point, and an ending point, return a path that solves the maze.

Example:

- Input:

  ```
  [
    '##### #',
    '#     #',
    '# #####',
    '#  ####',
    '## ####',
    '##  ###',
    '###   #',
    '##  ###',
    '#  ####',
  ]
  ```
